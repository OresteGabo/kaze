# Kaze Diagrams

This folder contains PlantUML source files for early architecture and data-model planning.

Files:
- `venue-map-domain.puml`: core class diagram for reusable venue maps, layout planning, and access control
- `conference-booking-erd.puml`: database-oriented entity diagram for conference/wedding venue reservations
- `kaze-pass-access-flow.puml`: sequence diagram for reservation/payment to Kaze Pass-gated entry

These diagrams are planning artifacts and should evolve with the implementation.
